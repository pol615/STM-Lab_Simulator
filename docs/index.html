GUI

Element Interaction

You may interact with elements if you hover over them. Once you hover over them, they change from their original color to red and will display information about that element (name, assigned values if it is a weigth, dynamometer force, etc.).

Once the element turns red, you shall be able to translate the element or pop up a menu using the right-click button from your mouse

Panning

You can pan an element as part of an element-interaction when the element has turned red by pressing left-click button down, move it to the desired place and lift left-click button up.

You can pan the canvas/board positioning the mouse on an empty space and by pressing left-click button down, move it to the desired place and lift left-click button up. Note that this option is a little bit buggy and you may preffer to intereact with the zoom in and zoom out option.

Zooming

You may want to zoom in by centering the mouse around the area you want to zoom in and scrolling the scroll-wheel up.

You may want to zoom out by centering the mouse around the area you want to zoom out and scrolling the scroll-wheel out.

You may want to reset the view by positioning the mouse in an empty space and using the right-click button to pop up a menu and select reset view.

You may want to fit the zoom to the current view by positioning the mouse in an empty space and using the right-click button to pop up a menu and select zoom frame. (NEW FEATURE)

Final Remarks

If you want to reset the app simulation you have to push the Reset App button. Note that the progress you have done will be lost.

Once the mechanical system has been set, you will be able to run the simulation. Note that if weights are note set or the system is not fixed, you will obtain an unexpected behaviour and luckily obtain a CUSTOM error. To run the simulation push the Simulate button.

The dropdown menu so-called Parent Element should refeer always to the element in which you have attached your loads and fixed. E.g., in the video below the element Lever is the one being fixed and applied loads onto it. If instead you had selected the pulley or weights, the dynamic rigid body movement would be solved there and that would be non-sensical since the they are not fixed.

After running a simulation and checking that the simulator responds back (note: you can use "1+1" or other operation in the command window to check the simulator has ended plotting, and if it does not return "2" it means it still is running something in background). You can click on Reset Simulation button to reset the mechanical system and elements to their state prior running the simulation.

If you may find that the simulator is running slower than usual, you may want to wait a little or in case there is no major upgrade, you should perform a Hard Reset. This procedure is simply to logoff and login back to matlab online so it may kill any unwanted or background process you had during your session and which may be slowing your simulation. This procedure is highly recommended once you start experiencing slowness and buggy display.

Basics

Fixation

You can set a fixation by interacting with an element (hover over an element and wait until the element has turn red) using the right-click button, waiting until a menu pops up and selecting the Fix menu and Add submenu option.

You can unset a fixation by interacting with an element (hover over an element and wait until the element has turn red) using the right-click button, waiting until a menu pops up and selecting the Fix menu and Erase submenu option.

Note that the different elements have a different way to set or unset the fixations, this is particularly explain in the ELEMENTS subsection inside the BASICS section.

This video features:

  • Fix your element through a hole.
  • Unfix your element through a hole.

Joint/Connection

You can set a connection/joint by interacting with an element (hover over an element and wait until the element has turn red) using the right-click button, waiting until a menu pops up and selecting the Con menu and Add submenu option.

You can unset a connection/joint by interacting with an element (hover over an element and wait until the element has turn red) using the right-click button, waiting until a menu pops up and selecting the Con menu and Erase submenu option.

Note that the different elements have a different way to set or unset the connections/joints, this is particularly explain in the ELEMENTS subsection inside the BASICS section.

This video features:

  • Set up a joint to connect links to your element.
  • Erase a joint.

Link

You can create a link between two conections by interacting with a created connection using the right-click button on to the green area spawn once the connection was created, waiting until a menu pops up and selecting the Linking menu and Start or End submenu option depending if you want to start or close a link between two connections.

You can erase a link by interacting with the so-called link (hovering over the link) by pressing the right-click button, waiting until a menu pops up and selecting the Erase option.

Note that the different elements have a different way to SET the links, this is particularly explain in the ELEMENTS subsection inside the BASICS section. However the erasing procedure is exactly the same since it is performed over the link itself.

This video features:

  • Link two connections.
  • Erase a link.

Elements

Lever

The Lever element has two options: Fix and Con. Since the Lever has multiple holes whenever Adding or Erasing a Fixation or Connection it will ask/expect you select which whole you want to connect/disconnect or fix/unfix by placing your mouse over the hole region and using left-click button to confirm it.

The procedure on how to Fix or Connect is by hovering over the lever (not the holes) wait until it has turned red and push the right-click button and select Add or Erase options from one of the two main options from the spawn pop-up menu.

This video features

  • Fixation : Select which hole to FIX or UNFIX
  • Connection : Select which hole to CONNECT or DISCONNECT

Pulley

The Pulley element has two options: Fix and Con.

To fix/unfix the Pulley, hover over it and wait until has turned red. Then push right-click button and wait until the pop-up menu spawns. Select Fix menu option and then either Addor Erase. The Pulley will expect you to select the unique hole it has by placing your mouse over the hole region and using left-click button to confirm it

To connect the Pulley, hover over it and wait until has turned red. Then push right-click button and wait until the pop-up menu spawns. Select Con menu option and then Add. The Pulley will create two green spots and it will ask you in which side you want to hung the weights. To select the desired connection side you have to place your mouse over the desired conection region and use left-click button to confirm it

To disconnect the Pulley, hover over it and wait until has turned red. Then push right-click button and wait until the pop-up menu spawns. Select Con menu option and then Add. It will automatically erase the connection.

Note that a connection should never be erased first if there are still links present. Do follow always erase the link first.

This video features:

  • Fixation : Select which hole (just one)
  • Connection : Select which side of the pulley to hang the weight

Weight

The Weight can only be connected/disconnected by hovering over it and waiting until it turns red, then pressing right-click button and waiting until the menu pops up and select Con option and then Add or Erase and it will automatically connect or disconnect the Weight.

The Weight value can be set by hovering over it and waiting until it turns red, then pressing right-click button and waiting until the menu pops up and select Set weight. A message should prompt in Command Window asking the total of mass the user wants to input (It has been seen a strange behaviour in which the message is not shown but still awaits for the user to insert the value in the command window). The value input can be seen now if you hover the Weight right to its name.

This video features:

  • Connection : Automatically selects the unique whole
  • Set weight : Prompts a message to introduce the weight (mass) in kg

Dynamometer

The Dynamometer can only be connected/disconnected by hovering over it and waiting until it turns red, then pressing right-click button and waiting until the menu pops up and select Con option and then Add or Erase and it will automatically connect or disconnect the Dynamometer.

This video features:

  • Connection : Automatically selects the unique hole
